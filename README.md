Project Description:

This project is a furniture-based e-commerce website built from a provided Figma design.
I carefully recreated the entire UI by matching the layout, typography, and styling to closely resemble the original design. The project was built completely from scratch, focusing on clean structure and responsiveness.
One of the key highlights of this project is the use of SCSS for styling. Since it was my first time working with SCSS, this project also served as a great learning experience in writing modular and maintainable styles.
The website is fully responsive and optimized for mobile view, ensuring a smooth user experience across different screen sizes.

Features :

1. Pixel-perfect UI based on Figma design
2. Fully responsive (mobile-friendly)
3. Built from scratch (no templates)
4. SCSS used for styling (modular & clean code)

Tech Stack :

HTML
SCSS
JavaScript
